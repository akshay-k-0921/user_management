# User Management System

## Features
- JWT Authentication
- Profile Management
- Secure CRUD for Tasks
- Vanilla JS Frontend

## Setup
pip install -r requirements.txt  
python manage.py migrate  
python manage.py runserver  

## Test command
python manage.py test
